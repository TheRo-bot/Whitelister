WHITELISTER WINDOWS README


DEPENDENCIES TO RUN:

Java
https://java.com/en/download/
Python3
https://www.python.org/ftp/python/3.7.4/python-3.7.4-macosx10.9.pkg

To compile this script, run the Compiler.bat application.
From script compilation, three files should appear in the Whitelister file:

TeamMaker
Creates Team files for the Whitelist program to read and understand.
Supplementary program.


Whitelist
Creates a whitelist file to be uploaded to a server via Uploader.
Upon opening a team file, select the teams you wish to use


Uploader
Attempts to upload a whitelist file to a server. 
Requires an internet connection to run.
